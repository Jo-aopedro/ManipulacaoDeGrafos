# PROJETO DIDÁTICO DE MANIPULAÇÃO DE GRAFOS

A base do programa foi toda desenvolvida pelos professores, ficando ao meu 
crédito a parte de desenvolvimento e implementação dos algorítmos de manipulação.

O programa conta com as funções de:

_Verificação se o grafo é Euleriano;
_Verificação se o grafo é Unicursal;
_Verificação se o grafo é Arvore;
_Listagem dos Pares Ordenados;
_Busca em profundidade;
_Busca em largura;
_Cálculo de caminho mínimo;
_Cálculo de Arvore Geradora Mínima (AGM);
_Cálculo de Número Cromático.

Ferramentas disponíveis:

_Exibir peso das arestas
_Gerar pesos Aleatórios
_Completar grafo
_Limpar Cores

Modo de Uso:

_CLICANDO sobre a tela aberta SÃO CRIADOS os VÉRTICES do grafo,
_O VÉRTICE ja EXISTENTE quando recebe um CLIQUE, é SELECIONADO e clicar sobre OUTRO 
que também ja existe CRIA uma ARESTA entre eles
_Para que as Arestas tenham PESO DIFERENTE de 1 e estes sejam EXIBIDOS na tela,
é necessário a seleção das ferramentas EXIBIR PESOS E GERAR PESOS ALEATÓRIOS
respectivamente, ANTES de iniciar
_A Ferramenta COMPLETAR GRAFO cria ARESTAS AUTOMÁTICAMENTE entre todos os vertices
existentes
_Criado o Grafo, é só selecionar algorítimo desejado na aba ALGORÍTMOS para a execução
_Alguns Algorítmos colorem vértices e arestas, o que é desfeito automáticamente
selecionando outro algorítmo, porém as cores podem ser removidas pela ferramenta
LIMPA CORES
